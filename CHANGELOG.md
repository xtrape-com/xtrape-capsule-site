# Changelog

Notable changes to the public Xtrape Capsule documentation site.

## [Unreleased]

- Continuous polish of public docs in preparation for v0.1.0 Public Preview.

## [0.1.0] - 2026-05-05

Initial public scaffold of `xtrape-capsule-site`.

### Added

- VitePress 1.6 site with custom homepage (Hero + Public Preview banner + ASCII architecture diagram + Quick Start + Developer SDK + Editions + Use Cases + Roadmap).
- Top nav and per-section sidebars: Guide, Concepts, Opstage CE, Agents, Contracts, Use Cases, Editions, Roadmap, GitHub.
- Concept pages: Capsule Service, Opstage, Agent, Agent Registration, Management Contract.
- Guide pages: Quick Start, Install Opstage CE, First Capsule Service, Demo.
- Opstage CE pages: Overview, Docker Deployment, Configuration, Admin UI, Backup and Upgrade.
- Agent pages: Node Embedded Agent, Action Model, Health Reporting, Config Reporting.
- Contracts pages: Overview, Manifest, Health, Actions, Errors.
- Use Cases: Integration Services, Playwright Workers, Account Pool, AI Agent Runtime.
- Editions: CE, EE, Cloud.
- Security: Overview (with safe-deployment checklist), Token Model, Agent Security.
- Releases: v0.1.0 draft notes.
- FAQ, Glossary, Legal, Roadmap.
- Local search via VitePress.
- GitHub Actions workflow deploying to GitHub Pages on push to `main`.
- LICENSE (Apache-2.0), NOTICE, CODE_OF_CONDUCT.md, SECURITY.md, CONTRIBUTING.md.
